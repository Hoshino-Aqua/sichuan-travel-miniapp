# 🏔 四川旅游邀请函小程序

一款基于微信小程序开发的旅游主题邀请函应用。  
通过邀请函展示、景点推荐、地图定位、在线报名等功能，推广四川旅游资源，提升用户互动体验。

---

## 📌 项目简介

本项目以“四川旅游邀请函”为主题，结合微信小程序开发技术，打造一个集 **景点展示 + 地图定位 + 活动报名 + 成功反馈** 于一体的旅游宣传类小程序。

系统围绕“邀请—浏览—定位—报名—反馈”完整流程设计，结构清晰、功能模块化，适合作为微信小程序综合实战项目。

---

## 🎯 项目目标

- 推广四川优质旅游资源
- 提供沉浸式邀请函体验
- 实现景点定位与地图跳转功能
- 完成活动报名流程
- 强化用户交互体验与视觉反馈

---

## 🛠 技术栈

- 微信小程序原生开发
  - WXML
  - WXSS
  - JavaScript
- 微信小程序 API
- 微信开发者工具
- 腾讯地图接口
- 本地数据模拟（JSON 数据）

---

## 🏗 系统功能模块

### 1️⃣ 欢迎页面

- 背景图沉浸式展示
- 动画效果（淡入 + 缩放）
- “开始使用”按钮跳转首页
- 提升首次进入体验

---

### 2️⃣ 首页（景点推荐）

- 轮播图展示四川风光
- 横幅主题宣传
- 推荐景点列表
- 点击跳转景点详情定位页

推荐景点包括：

- 九寨沟
- 峨眉山
- 都江堰
- 成都
- 稻城亚丁
- 青城山
- 海螺沟

---

### 3️⃣ 景点定位页

- 接收首页传递的经纬度
- 使用 `<map>` 组件显示地图
- 自定义 marker 图标
- 点击跳转腾讯地图查看具体位置
- 支持定位当前景区

---

### 4️⃣ 报名页面

- 用户头像上传
- 昵称填写
- 手机号填写
- 表单提交
- 页面跳转

---

### 5️⃣ 报名成功页面

- 动态 GIF 动画展示
- 视频播放功能
- 营造庆祝氛围
- 强化用户参与感

---

## 🗄 数据结构设计

### 景点数据（spots）

字段示例：

- name（景点名称）
- description（景点介绍）
- image（图片地址）
- latitude（纬度）
- longitude（经度）

采用本地 JSON 数据模拟数据库结构，支持后续扩展为云数据库版本。

---

## 🔄 用户使用流程

1️⃣ 打开小程序  
2️⃣ 浏览欢迎页面  
3️⃣ 进入首页查看景点  
4️⃣ 点击景点查看地图定位  
5️⃣ 进入报名页面填写信息  
6️⃣ 提交后进入成功反馈页面
